# ~/.gemini

This directory contains configuration and reusable engineering guidance for AI coding assistants.

## Structure

```text
.
├── README.md          # This file
├── GEMINI.md          # Gemini bootstrap
└── esper/             # Esper AI Engineering Operating System
```

## Components

### GEMINI.md

The bootstrap prompt for Gemini.

It initializes the assistant and directs it to Esper.

### esper/

Esper is a modular engineering operating system for AI coding assistants.

It contains reusable engineering knowledge organized into:

- Prompt Specifications
- Workflows
- Engineering Principles
- Checklists
- Templates

See `esper/index.md` for the complete specification.
