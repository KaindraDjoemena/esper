# Testing Checklist

## Purpose

Ensure test coverage is sufficient and tests are maintainable.

## When to Use

- writing tests
- reviewing test coverage

## Testing Verification

- [ ] Happy path covered
- [ ] Failure paths covered
- [ ] Edge cases covered
- [ ] Regression risk evaluated
- [ ] Existing tests remain valid
- [ ] Test readability maintained
