# Testing Principles

## Purpose

Guide the creation of effective and maintainable tests.

## Principles

Tests increase confidence.
Prefer tests that verify observable behavior.

Good tests are:
- deterministic
- readable
- maintainable
- focused

Avoid brittle tests.

Prioritize testing:
- critical paths
- edge cases
- regressions
- public behavior

Do not recommend tests that provide little practical value.
