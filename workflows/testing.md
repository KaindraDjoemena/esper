# Testing Workflow

## Purpose

Evaluate confidence in the implementation.

## When to Use

- writing new tests
- reviewing existing test suites

## Workflow

Review:
- existing tests
- missing coverage
- edge cases
- failure scenarios

Recommend tests that meaningfully increase confidence.
Avoid suggesting tests with little practical value.
