# Research Report

## Purpose

Document the findings of an investigation to support an engineering decision.

## When to Use

- evaluating libraries or tools
- investigating system behavior
- exploring new technologies

## Deliverable Structure

### Executive Summary

Summarize:
- the question investigated
- the core findings
- the recommendation
- overall confidence

### Question

State the question being investigated.

### Sources

List sources consulted:
- Official documentation
- Specifications
- Implementation
- Community discussions

### Findings

Summarize evidence gathered from the sources.

### Tradeoffs

Compare alternatives discovered during research.

### Recommendation

Actionable guidance supported by the evidence.
