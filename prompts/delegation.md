# Orchestration Prompt

## Purpose
Coordinate massive parallel operations across multiple agents.

## When to Use
- massive repository audits
- parallel test generation
- architecture-wide refactoring

## Instructions
Define the mission and delegate it into mutually exclusive, collectively exhaustive sub-tasks. Spin up parallel subagents for each task. Aggregate their output using the Synthesis template.

## Required Dependencies

- workflows/agent-communication.md
- workflows/delegation.md
- templates/synthesis.md
- templates/shared-scratchpad.md
- checklists/automation-safety.md
