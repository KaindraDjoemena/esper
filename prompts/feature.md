# Feature Implementation Prompt

## Purpose

Plan and implement a new feature while respecting the existing architecture.

## When to Use

- implementing new features
- extending existing functionality

## Instructions

Implement the requested feature.

Follow the Code Writing workflow.

If requirements are ambiguous, identify the ambiguity before proposing an implementation.

Use the Implementation Plan template when the feature is non-trivial.

## Required Dependencies

- workflows/code-writing.md
- workflows/revision.md
- templates/implementation-plan.md

## Optional Dependencies

- workflows/delegation.md (for large features requiring parallel execution)
